# AI and app's permission

## Project Description
This project studies how AI affects mobile application permissions by analyzing multiple versions of phone apps.

## Methods
- Static analysis of APK files using apktool
- System-level inspection using ADB

## Data
Permission datasets are stored in the /data and /processed folders.

## Key Findings
- Permissions increase over time
- AI-related features introduce more input-based permissions
- Different AI paradigms lead to different permission patterns
